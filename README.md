# go-epp

A production-grade Extensible Provisioning Protocol (EPP) SDK for Go.

## Status

🚧 Under active development

## Goals

- RFC 5730 - EPP Core
- RFC 5731 - Domain Mapping
- RFC 5732 - Host Mapping
- RFC 5733 - Contact Mapping
- RFC 5734 - TCP Transport
- Registry Extensions
- Production Ready
- Well Tested
